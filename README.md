## Descripción del proyecto

Este proyecto tiene como objetivo comparar dos enfoques de programación para resolver un mismo problema: el registro y visualización de información básica de una mascota.

Los datos manejados son:

- Nombre
- Especie
- Edad

## Programa Tradicional

La solución tradicional utiliza:

- Variables
- Funciones
- Entrada de datos por teclado

No utiliza clases ni objetos.

## Programa Orientado a Objetos

La solución orientada a objetos utiliza:

- Clase Mascota
- Objetos
- Atributos
- Métodos
- Abstracción

Se crean dos objetos de la clase Mascota y se ejecutan sus métodos.

## Reflexión

La Programación Tradicional organiza el código mediante funciones y variables, siendo adecuada para programas pequeños. Sin embargo, cuando el sistema crece puede resultar más difícil de mantener.

La Programación Orientada a Objetos permite representar entidades del mundo real mediante clases y objetos, facilitando la reutilización del código, la organización de la información y el mantenimiento de programas más grandes. Por esta razón, la POO es ampliamente utilizada en el desarrollo moderno de software.
